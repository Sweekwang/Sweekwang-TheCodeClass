Data Science

Data science is an field focused on getting knowledge from data sets. The field encompasses analysis, preparing data for analysis, and presenting findings. Important Data Science skills are statistical programming language, like R or Python but over here, we only teach you Python. The most fundamental skill is statistics which is the core of Data Science. Lastly, you also need to know Data Wrangling, Data Visualization, and Machine Learning.


TheCodeClass, will teach you Python, Data Wrangling, Data Visualization, Machine Learning and some statistics knowledge.

To get started. Learn Basic Python 1st. After that we will teach you Data Wrangling and some statistics knowledge in Data analysis using NumPy and Pandas. Then Data Visualization and Machine Learning after that.

Data Science

We teach you from the very basic. First, we teach you about basic Python as it is one of the important language for data science and provide tools for statistics. After that, you will use NumPy and Pandas to learn about some data structure like the one you see in spreadsheet or excel. Then we will plot some graphs and use machine learning models.
