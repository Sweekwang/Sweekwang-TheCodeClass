About Basic Python Course

(and Prerequisite)

Basic Python course caters for beginners to start learning your first programming programming language and for intermediates to re-cap their Python knowledge for free. Before learning data analytics and data science tools or becoming an Python developer, you need to understand the basic first.

Prerequisite: No Prerequisite needed. Just require your interests.

After this course: for students who wants to go into Data Science and Data Analytics, continue learning NumPy > Pandas > Data Visualization > Mechine Learning.

Students who have interests in becomeing Python Developer, learning to build web and etc, continue learning Advanced Python > HTML & CSS > Web Development.
